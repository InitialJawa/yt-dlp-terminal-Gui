# yt-dlp Terminal GUI

Terminal UI untuk **yt-dlp** — download video YouTube & lainnya pake menu gampang.

## Install

```bash
npm install -g https://github.com/InitialJawa/yt-dlp-terminal-Gui
```

Atau langsung pake `npx` tanpa install:

```bash
npx https://github.com/InitialJawa/yt-dlp-terminal-Gui
```

## Usage

```bash
ytdl
```

## Fitur

- **Pilih resolusi** — 1080p, 720p, 480p, Audio MP3, atau Best (auto)
- **Pilih folder** — Downloads, Videos, Desktop, Manual path, atau Browse
- **Auto-install** — cek Python & yt-dlp, kalau belum ada langsung install
- **Progress bar** — liat status download real-time

## Syarat

- Windows 10+
- Internet (buat install & download)
